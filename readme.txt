Integrate your Wordpress with a simple vote plugin with smileys!

== Description ==

This plugin allows you to set a votation for the post and pages.


== Features ==

* Auto insert in content (can be disabled).
* Filter by post and page.
* Count totals and calcule the percent of votes.
* 3 Type of votes: Good, Neutral and Bad.
* See the votes on the admin panel.
* Widget to insert whenever you want.
* Widget to see the top of rated post.
* Shortcode to inser whenever you want.
* Two styles for Horizontal and Vertical so you can put where you want!.


== Installation ==

This section describes how to install the plugin and get it working.

= Installing the Plugin =

*(using the Wordpress Admin Console)*

1. From your dashboard, click on "Plugins" in the left sidebar
2. Add a new plugin
3. Search for "Simple Vote me"
4. Install "Simple vote me".
5. In settings page you can set a few options!

*(manually via FTP)*
1. Upload the 'gt-simple-vote-me' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In settings page you can set a few options!


== Frequently Asked Questions ==

= Can I set a custom CSS? =

Sure!, Go to admin panel and in the option page for "Simple vote me" you can add your custom CSS.


= How to modify if I want Horizontal or Vertical? =

You can set it using the Widget (in the options Widget) or in the shortcode [simplevoteme type="v"] for vertical, or echo [simplevoteme type="h"] for horizontal.


= What about validation errors, will it still maintain my WP theme? =

The plugin uses minimum CSS and inserts valid HTML when necessary.
